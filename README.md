# PO+PLE 포커 커뮤니티 웹사이트

## 배포 설정

이 웹사이트는 **정적 웹사이트**입니다.

### Replit 배포 설정:
- **배포 타입**: Static Deployment
- **Public Directory**: `/` (루트 디렉토리)
- **Build Command**: 없음 (정적 사이트)
- **Run Command**: 필요 없음 (정적 사이트)

### 파일 구조:
- `index.html` - 메인 페이지
- `404.html` - 에러 페이지
- `*.png` - 이미지 파일들
- `*.py` - 개발 서버용 (배포에서는 사용 안 함)
